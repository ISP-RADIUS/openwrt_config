# OpenWRT Configuration Manager

## Introduction

## Usage

Create directories for the configuration system and database:

```
mkdir /mnt/config
mkdir /mnt/config/database
```

Clone Repository
```
cd /mnt/config
git clone https://github.com/ngardiner/openwrt_config openwrt
```

### Updating

```
cd /mnt/config/openwrt
git pull
```
